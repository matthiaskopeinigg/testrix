# Diavolo

Display typeface for the home landing **Testrix** title.

- **Designer:** [Figuree Studio](https://figureestudio.com/)
- **Source:** [DIAVOLO on 1001 Fonts](https://www.1001fonts.com/diavolo-font.html) (free for commercial use per site license; see `Read Me` in the original download zip if you replace this file)
- **Bundled file:** `Diavolo.woff` (webfont)

If you update the font, keep the filename or adjust `src/styles/_font-diavolo.scss`.
