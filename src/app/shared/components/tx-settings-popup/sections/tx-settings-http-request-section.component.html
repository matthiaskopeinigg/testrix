Request

Default HTTP behavior when opening and sending requests. Applied globally until per-request overrides exist.

Defaults

Used when the current response tab cannot be shown. Your last tab (for example Timeline) stays selected when you send again.

When a request URL has no scheme, Send prepends the default scheme and can add www. for bare domains (e.g. google.at → https://www.google.at).

When enabled, Send adds a Content-Type header from the body editor mode and payload (for example JSON in a Text body). Add your own Content-Type header to override.
