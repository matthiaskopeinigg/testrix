Start Testrix from the Start menu or your desktop.

Installer help

This installer copies Testrix to your computer and creates the shortcuts that match your operating system.

Install scope

  • Only for me — installs under your user profile. No administrator rights required.
  • Everyone on this PC — installs to a system-wide location. Will prompt for administrator rights.

Uninstall later

Open your OS app manager, find Testrix, and choose Uninstall.

Other platforms

Testrix ships installers for Windows, macOS and Linux. If you also use another OS, grab the matching artifact from the Releases page:

  • Windows — Testrix Setup.exe
  • macOS — Testrix Setup.dmg
  • Linux — Testrix Setup.AppImage

Troubleshooting

If the installer says the payload is missing, download the installer again or rebuild with npm run electron:pack.
