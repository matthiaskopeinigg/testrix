Uninstaller help

This uninstaller removes Testrix from your computer. The application stays running until you click Uninstall, so you can cancel at any time before that.

What gets removed

    User data

    Tick Also remove user settings and data to additionally delete your local configuration folder. Workspaces and request collections live there.

    Other platforms

    Testrix also runs on macOS and Linux. If you uninstall on another OS, the matching steps are listed at the Releases page:

    • Windows — Settings → Apps → Installed apps, or run uninstall.cmd from the install folder.
    • macOS — drag Testrix.app to the Trash, or run this uninstaller bundled inside the .app.
    • Linux — apt remove testrix for .deb, or delete the AppImage and its .desktop entry.
